Natural Language Processing (NLP) is a field of Artificial Intelligence and computational linguistics that helps computers understand, analyze, and generate human language. Text data, such as emails, social media posts, customer reviews, news articles, medical reports, and legal documents, is one of the most common forms of unstructured data. Since raw text cannot be directly used in machine learning models, it must first be converted into a structured numerical format through various NLP techniques.

pd.merge(df1, df2, on='Key'): This function is used to combine two DataFrames by matching rows based on a common column called the key.

how='inner': This merge option keeps only the rows where the key is present in both DataFrames.

how='outer': This option keeps all rows from both DataFrames and fills unmatched values with NaN.

how='left': This option keeps all rows from the first (left) DataFrame and only the matching rows from the second (right) DataFrame.

how='right': This option keeps all rows from the second (right) DataFrame and only the matching rows from the first (left) DataFrame.

df.columns: Used to display the names of all columns available in a DataFrame.

df.shape: Returns the size of the DataFrame in terms of the number of rows and columns.

df.info(): Provides a summary of the DataFrame, including column names, data types, non-null values, and memory usage.

df.describe(): Generates statistical details such as count, mean, standard deviation, minimum, and maximum values for numerical columns.

print(df): Displays the complete DataFrame output for checking and understanding the dataset.
